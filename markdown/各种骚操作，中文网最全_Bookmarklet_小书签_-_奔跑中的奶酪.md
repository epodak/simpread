了解小书签

什么是小书签？

小书签的英文名 **Bookmarklet**，是由 **Bookmark**（书签） 和 **Applet**（小程序）组合而来，叫做 “**书签小程序”**。

1、从形式上来看
--------

小书签的本质就是**书签**。

它是 “**Javascript 之父**” **Brendan Eich** 为了开发方便而特意设计。

可以将一段带有特定功能的 **JavaScript** 代码保存在收藏夹里，需要的时候，**点击**就可以实现代码的功能。

也就是说，小书签其实和一般的书签一样，**都是书签**。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlz86q5oj22a41fuqhr.jpg)

而 **http:** 这样的前缀，其实是**浏览器协议**。

它的作用是告诉浏览器接下来要做什么，比如还有 **https:、ftp:、file:、mailto:、ed2k:、magnet:** 等诸多协议。

它们都可以保存到 “**书签**”，点击就会执行相应的命令。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlz953o5j21hc0u0asv.jpg)

2、从作用上来看
--------

小书签的本质是 **JavaScript** 脚本。

小书签，其实可以看作是一个**独立的、原生的、简单的**脚本或者扩展。

扩展的优势在于**有良好的交互界面**，脚本的优势在于**可以自动运行**，而小书签的优势则是 “**没有冗余设计**”。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2e2e9kw3ij21hc0u01kx.jpg)

就好比用户想要吃 “**黄瓜**”，扩展给的是 “**黄瓜炒肉**”，油猴脚本给的是 “**凉拌黄瓜**”，而小书签给的是 “**生黄瓜**”。

也就是说，扩展安装多了，会占用较多的内存资源，油猴脚本安装多了，也会影响浏览器速度。

而且，不同扩展和脚本之间，还会有不兼容和相互打架的问题。

**而小书签则完全不会有影响。**

小书签演示

下面是部分小书签的演示说明，具体可看表格说明。

1、生成二维码（qr）
-----------

这个小书签可以让你一键生成当前网页的二维码，而且是本地生成，零延迟。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1hkytvq78vtj221c1cgay1.jpg)

2、油猴脚本下载（gf）
------------

曾经有一个可以显示当前网站可用油猴脚本的脚本，叫 “**Userscript+**”。

但这个脚本非常大，会严重拖慢网页加载速度，效果其实并不好，所以作者后续也删除了这个脚本。

其实用小书签可以一键查询，效果也一样。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36k9i7wjvj228q1fw7tn.jpg)

3、各种封面下载（CB）
------------

如果想下载 U2B、B 站、公众号的封面，其实不用安装脚本，也不用到各种网站上去提取，小书签也能一键下载。

快捷键分别为大写的 **CY CB CW**。

也就是 **Cover for Youtube，Cover for Bilibili，Cover for Weixin** 的简写。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36k9mus84j228q1fwhdt.jpg)

4、编辑当前网页（ed）
------------

眼见不一定为真，截图也一样，这个小书签很能证明这一点。开启小书签后即可任意编辑网页。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36k9rwoa4g221c1c4asd.gif)

5、解除网页限制（re）
------------

这个功能可能是呼声最高的一个小书签了。

不需要任何扩展，不需要任何脚本，小书签也一样能解除网页限制，而且效果更好。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36k9vbu9ej228q1fwb2b.jpg)

如果是一些[**顽固的网页**](https://nujuo8y1qx.feishu.cn/docs/doccnGbXj47lGM1iqHk7L1UyW8p)，则可以用 “**解除 CSP 限制**” 的方法，快捷键 **RE**（大写），也就是按住 **Shift** 键按 **re** 键。

两个小书签配合使用，能解决几乎 100% 的网页限制问题。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1hkytxydvkjj228c1echdt.jpg)

6、弹窗打开视频（pp）
------------

浏览器的 “**画中画**” 功能不支持弹幕，想要支持弹幕需要安装比如 **Popup Window** 这样的扩展。

小书签 “**弹窗打开视频**” 也可以实现同样的功能。

至于 “**置顶播放**” 的问题，可以使用鼠标手势软件 **Mouseinc** 划 **M** 型来置顶窗口。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36k9ymd9vj228q1fwu0y.jpg)

7、复制网页 Markdown（cc）
-------------------

在复制地址栏链接时，如果出现 **26** 个英文字母之外的字符，那么链接就会被转义为 %20 之类的 “**乱码**”。

使用快捷键 **cc**，不但可以一键复制网页 Markdown 链接，而且还能保持**原链接**。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h36ka3easbj228q1fwb29.jpg)

8、百度边栏搜索（bb）
------------

平时在看视频时，如果想搜索某个关键词，使用地址栏搜索，则必然会打断当前进度。

使用快捷键 **bb**，则可以在侧边栏里进行搜索。

这里有三种情形。

一是，如果输入的是网页链接，那么就会在侧边栏**打开该网页**。

二是，如果输入的是文字内容，那么就会在侧边栏打开**百度搜索**。

三是，不输入任何内容，直接按回车，那么打开的就是**当前网页**。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/008vxvgGgy1h8wovwnjd6j31bw0u045w.jpg)

使用小书签

有了小书签只是第一步，真正让小书签从 “**鸡肋**” 变成 “**神器**” 的，是它的使用方法。

一、点击调用小书签
---------

第一种方法最直观，就是将 **javascript**: 开头的链接，拖入到浏览器书签工具栏，然后点击运行。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzba9mfj225c1f6u0i.jpg)

二、关键字调用小书签
----------

第二种方法，是给小书签添加**关键字**。

**Firefox** 可以使用快捷键 **Ctrl+Shift+O** 打开 “**书签管理器**”，然后给小书签直接添加 “**关键字**”。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzbrif1j21ug16gx3o.jpg)

**Chrome / Edge** 等浏览器，则需要将小书签保存到 “**搜索引擎**” 里，然后添加相应的**关键字**。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzc9fdaj225c1f6qh0.jpg)

添加后，在地址栏里输入**关键字**，那么就会调用小书签。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzct24lj22621eq7nt.jpg)

PS：如果你的 **Firefox** 设置了地址栏链接在**新标签页打开**，那么需要按住 **Alt** 键（Mac 用 **Cmd** 键）点击启用。

三、快捷键调用小书签
----------

真正让小书签使用体验拉满的方式，是**快捷键**。

**ShortKeys** 是一款快捷键扩展 ，它除了可以给浏览器的各项功能**增加快捷键**，还可以通过快捷键**调用小书签**。

它的**工作原理很巧妙**。

就是模拟点击某个书签，也正是因为这个原因，即使你有再多的小书签，**也丝毫不会影响浏览器速度**。

你可以在扩展的 “**设置页面**” 里指定小书签的**快捷键**。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzdb9sjj225c1f6qm5.jpg)

**四、添加快捷键**
-----------

如果你还想用更快的方法来给小书签添加**快捷键**，那么可以用通过**代码指令**来导入。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/e6c9d24egy1h2g3aigyjdj21aa0u0whh.jpg)

**添加一个小书签快捷键：**

{"key":"a a","label":"aaaa","bookmark":"aaaa","action":"openbookmark","sites":"","sitesArray":[""]}

**添加多个小书签快捷键：**

[  
{"key":"a a","label":"aaaa","bookmark":"aaaa","action":"openbookmark","sites":"","sitesArray":[""]},  
{"key":"b b","label":"bbbb","bookmark":"bbbb","action":"openbookmark","sites":"","sitesArray":[""]},  
{"key":"c c","label":"cccc","bookmark":"cccc","action":"openbookmark","sites":"","sitesArray":[""]}  
]

**注意事项：**

1、**key** 是小书签的**快捷键**，**label** 和 **bookmark** 是小书签的**名称**，其它的不用修改。

2、key、label 和 bookmark 的值，结尾不要有**空格**，**空格**会导致调用失效。

3、如果有两个小书签使用了**一样的名字**，那么只有一个会被执行。

4、如果快捷键同时有 **bg** 和 **bgg**，那么 **bgg** 不会被执行。

5、最后一行的 **[""]}** 处，不能有**逗号**。

**五、兼容性问题**
-----------

以上小书签，使用扩展 **Shortkeys**，在 Chrome 浏览器上调用都没有问题。

但在 Firefox 上，一些小书签无法调用，目前已知的小书签包括：**as、hv、vv、iii、iib、lll** 等。

解决的方法是使用 “**Run JavaScript**” 来调用。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/e6c9d24egy1h2g4ohca1gj219d0u044b.jpg)

也就是将小书签的代码，粘贴到 “**JavaScript Code**” 输入框里。

不过，还需要将小书签的代码还原为**最原始的**状态，可以借助 [https://tool.oschina.net/encode?type=4](https://tool.oschina.net/encode?type=4) 来完成。

之后就可以正常调用了。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/e6c9d24egy1h2g5ph3z8ej219d0u0jxj.jpg)

**二、快捷键思路**
-----------

下面是奶酪的快捷键设置思路：

**1、使用单键连续点击。**

快捷键的设置，我选择了**单键连续点击**，可以在防止误触的同时，以**最快的速度**调用小书签。

比如快捷键 **qr**，就是依次按下 **q** 和 **r** 键。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/e6c9d24egy1h2esdlcoepj219c0u00xs.jpg)

区分大小写，**大写的**快捷键，也就是按住 **Shift** 键点击，通常是这个小书签的**备用选项**。

**2、快捷键为英文或拼音。**

快捷键的字母，我选择了小书签功能的**英文或拼音**，比如 **qr** 就是 **qr code** 的缩写，具体可以看列表清单。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzfhakuj225c1f6x3v.jpg)

**三、注意事项**
----------

**1、在浏览器内建页上无法使用。**

包括 “**新标签页**” 在内的所有浏览器内建页，无法通过快捷键的方法调用小书签。

**2、当聚焦点在输入框时无法使用**。

这时可以按下 **Tab** 键，将聚焦点移出输入框后即可使用。

**3、如果网页自带快捷键，那么也不会优先启用。**

B 站、U2B、GitHub 等网站，本身自带了**快捷键**，这时快捷键不会被优先启用。

**4、一些小书签在 CSP 网站上无法启用。**

如果网站开启了 [CSP 内容安全策略](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/CSP)，比如知乎、GitHub 等网站，那么调用了第三方脚本的小书签也将无法使用。

比如 “**阅读模式**” 小书签就调用了第三方脚本。

项目开源

很惭愧，就是做了一点微小的工作。

奶酪的三个 “**微小的工作**”，可能会让这项**远古神迹**，在中文互联网将 “**再次伟大**”。

工作一、最全小书签
---------

奶酪的「**小书签**」维护，是从 **2014** 年开始的。

此后有 **11** 次更新，然后每次更新都会集成在奶酪的 “**定制版浏览器**” 里，而这一次的更新 “**更是重量级**”。

奶酪原创、修复和收集的小书签，总计将近 **300** 个，中文网最全，没有之一。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlza7fmhj225c1f61ke.jpg)

这让小书签的可用性，从 **0** 到 **1**，实现了从无到有的突破。

工作二、形象化小书签
----------

同时，奶酪还给每个小书签都加上了**图标**。

这样一来，每个小书签就从原来的 “**一串代码**”，变成了**视觉化的、可记忆的**、而且还是**易于理解的**工具。

你可以在奶酪 [公众号](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1fr3v1xxt8ej205k05kt8p.jpg) 回复关键字 **i07** 获取奶酪精心设计的 **html** 小书签，然后导入浏览器。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/7a6a15d5gy1h2dlzaqqjnj225c1f6wob.jpg)

这让小书签的**易用性**，从 **1** 到 **10** 实现了指数级提升。

工作三、快捷调用小书签
-----------

而奶酪的第三个工作，是找到了真正快捷的**小书签调用方式**。

**[ShortKeys](https://github.com/crittermike/shortkeys)** 这款快捷键扩展 ，可以让我们以最快的速度调用小书签，而且丝毫不会影响浏览器速度。

比如快捷键 qr，就是依次按下 q 和 r 键。

![](https://cdn.cdnjson.com/pic.html?url=https://fc.sinaimg.cn/large/e6c9d24egy1h2esdlcoepj219c0u00xs.jpg)

这让小书签的**可及性**，从 **10** 直接彪升至 **100**，使用体验直接拉满。
